# Project: Cross-Snip

## Stack
- Electron + React + Vite
- Supabase (DB + Auth)
- Desktop app (Mac/Windows)

## Local Dev
- Run `npm run dev` to start
- Env vars in `.env` (loaded via direnv)

## Structure
- /src — React frontend
- /electron — Electron main process
- /dist — build output

## Conventions
- Never commit .env
